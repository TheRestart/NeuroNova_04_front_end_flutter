# Mesothelioma 헬스케어 앱 프로젝트

## 📋 프로젝트 정보

**프로젝트명**: Mesothelioma
**타입**: Flutter 멀티 유저 헬스케어 애플리케이션
**버전**: 1.0.0+1
**플랫폼**: Android (현재 설정 기준)
**개발 환경**: Flutter SDK ^3.9.2

---

## 🎯 프로젝트 목적

중피종(Mesothelioma) 환자의 건강 관리를 위한 종합 헬스케어 플랫폼으로, 세 가지 사용자 역할을 지원합니다:

- **환자(Patient)**: 진료 예약, 의사 검색, 건강 정보 관리
- **의사(Doctor)**: 환자 관리, 진료 일정 관리, AI 예측 데이터 확인
- **관리자(Admin)**: 사용자 승인, 전체 회원 관리

---

## 🏗️ 아키텍처 패턴

**Feature-Based Architecture (도메인 주도 설계)**

```
lib/
├── core/           # 전역 핵심 요소 (상수, 서비스, 공용 위젯)
├── data/           # 데이터 계층 (모델, Repository)
├── features/       # 기능별 모듈 (역할 기반 분리)
│   ├── auth/       # 인증 (로그인/회원가입)
│   ├── admin/      # 관리자 기능
│   ├── doctor/     # 의사 기능
│   ├── patient/    # 환자 기능
│   ├── home/       # 공용 홈 화면
│   └── common/     # 공용 페이지
└── main.dart       # 앱 진입점
```

## 📱 주요 기능

### 1. 인증 시스템 (Auth)
- 로그인 화면 (`SignInScreen`)
- 회원가입 화면 (`SignUpScreen`)
- 이메일/비밀번호 검증 (form_field_validator 사용)

### 2. 환자 기능 (Patient)
- 대시보드 홈 화면
- 카테고리별 메뉴 (검사 결과, 진단서 등)
- 진료 가능한 의사 목록
- SVG 아이콘 기반 UI

### 3. 의사 기능 (Doctor)
- 진료 일정 캘린더 (Syncfusion Calendar)
- 환자 카드 리스트
- AI 예측 요약 정보
- 일정 관리 기능

### 4. 관리자 기능 (Admin)
- 회원 가입 승인 페이지 (`ApprovalPage`)
- 전체 회원 목록 (`UserListPage`)
- 회원 상세 정보 (`UserDetailPage` - 미구현)

### 5. 공용 기능
- 역할별 하단 네비게이션 바
- 내 정보 페이지 (`MyInfoPage` - 미구현)
